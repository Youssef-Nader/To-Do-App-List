# Task Manager App

A simple React application to manage daily tasks.  
You can add, edit, delete, and mark tasks as completed.

## Features
- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Mark tasks as completed  
- View:
  - All Tasks  
  - Completed Tasks  
  - Incomplete Tasks  

## Tech Used
- React  
- React Router  
- Context API  

## How to Run
```bash
npm install
npm start
